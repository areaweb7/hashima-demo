# 中部第一輸送株式会社 岐阜羽島営業所 ウェブサイト

## 概要

中部第一輸送株式会社の岐阜羽島営業所のリニューアルオープンに伴う、営業所紹介ページです。

**サーキュラーエコノミー（循環型経済）**を強調し、環境保全事業としての側面をアピールする内容になっています。

### 主な特徴

- **サーキュラーエコノミーの強調**: 環境保全・資源循環型社会への貢献を前面に
- **豊富な画像プレースホルダー**: デザイナーとの相談用に50箇所以上の画像配置エリアを設定
- **レスポンシブデザイン**: PC、タブレット、スマートフォンに完全対応
- **モダンなUI/UX**: グラデーション、シャドウ、アニメーション効果を使用
- **SEO対策**: 「サーキュラーエコノミー」などのキーワードを戦略的に配置
- **地域密着**: 岐阜県羽島市を拠点とした地域貢献活動の紹介

## セットアップ方法

### ローカル環境での確認

1. ファイルをダウンロード
2. `index.html` をブラウザで開く

### Webサーバーへのアップロード

1. FTPまたはSFTPで全ファイルをアップロード
2. パーミッションを確認（HTML: 644, CSS: 644, JS: 644）
3. ブラウザでアクセスして動作確認

### WordPressへの組み込み（オプション）

固定ページに以下の手順で組み込み可能：

1. WordPressの「固定ページ」→「新規追加」
2. 「カスタムHTML」ブロックを追加
3. `index.html` の`<body>`内のコンテンツをコピー
4. テーマのカスタムCSSに`style.css`の内容を追加
5. テーマのカスタムJSに`script.js`の内容を追加

## カスタマイズガイド

### 画像の追加

画像を追加する場合は、以下の箇所を編集：

```html
<!-- ヒーローセクションに背景画像を追加 -->
<section class="hero" style="background-image: url('images/hero-bg.jpg');">
```

### 色の変更

`style.css`の`:root`セクションで色を変更：

```css
:root {
    --primary-color: #0066cc;  /* お好みの色に変更 */
    --secondary-color: #003d7a;
    --accent-color: #00aaff;
}
```

### 連絡先情報の更新

